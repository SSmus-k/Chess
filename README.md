## Overview

Chess is a **Python-based chess application** that combines a clear architecture with powerful AI capabilities, making it ideal for developers interested in game development, AI research, or educational projects.

### Why Chess?

This project provides a robust foundation for building and analyzing chess games, with core features including:

- 🧩 **Modular Design:** Clear separation of core components like game logic, AI, and user interface for easy understanding and customization.
- ⚙️ **Game State Management:** Reliable tracking of moves, validation, and undo functionality to ensure game integrity.
- 🤖 **AI Engine:** An intelligent opponent that evaluates moves using a depth-limited **negamax search with alpha-beta pruning**.
- 🎮 **Flexible Modes:** Support for human vs. AI, AI vs. AI, and other configurations to suit various use cases.
- 📦 **Dependency Control:** Ensures consistent environment setup with specified library versions via `requirements.txt`.

### Prerequisites

This project requires the following dependencies:

- **Programming Language:** Python

---

### Installation

Build Chess from the source and install dependencies:

1. **Clone the repository**

git clone https://github.com/SSmus-k/Chess

Navigate to the project directory

cd Chess

Install dependencies

pip install -r requirements.txt
Usage

Run the project with:

python {entrypoint}
Testing

Chess uses the {test_framework} test framework. Run the test suite with:

pytest
